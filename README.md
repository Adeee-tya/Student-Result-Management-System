# Student Result Management Portal

A robust, full-stack Academic Performance Tracking system built using a Python-based Flask backend engine and a relational MySQL database. The application features intuitive administrative controls for tracking enrollments, logging student evaluations, and dynamically compiling university transcripts.

## 🚀 Core Functional Modules
* **Dynamic Enrollment Desk:** Enrolls student profiles with custom fields tracking institutional metadata and active term semesters.
* **Academic Evaluation Console:** Logs subject evaluations, scoring tallies, and validation boundaries.
* **Real-Time Ad-Hoc SQL Terminal:** An interactive dashboard database console for testing ad-hoc queries securely through the web UI.
* **Automated Transcript Engine:** Joins schema components dynamically to compile student report sheets with automated conditional letter grade mappings (`A+` through `F`).

## 🛠️ Technology Stack
* **Backend:** Python, Flask, MySQL Connector API
* **Frontend:** HTML5, CSS3 Grid Layouts, Jinja2 Template Engine
* **Database Layer:** MySQL Workbench RDBMS (Third Normal Form Schema)
